# json_the_cat

## Instructions:

`npm init -y`

`npm install request`

`touch .gitignore && echo "node_modules/" >> .gitignore`

# Testing

`npm install mocha chai --save-dev`

Create a test directory and test file.

Update JSON file to use mocha for the test
```
  "scripts": {
    "test": "mocha"
  }
```

Run test using  `npx mocha`
